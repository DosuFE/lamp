# 🚀 LAMP — Learning & Assessment Management Portal

A full-stack Learning & Assessment Management Portal (LAMP) designed to manage courses, lectures, and CBT-based examinations in a modern and scalable way.


# Overview

LAMP is an online platform that allows:
- Students to enroll in courses and take exams
- Admins to manage courses, lectures, and tests
- Automated grading through a CBT (Computer-Based Test) system

Built with a scalable architecture using modern web technologies.


#✨ Features

# 👨‍🎓 Student Features
- User registration & login (JWT authentication)
- Browse and enroll in courses
- Access course lectures (restricted access)
- Take CBT exams with timer
- View results after submission



# 🛠️ Admin Features
- Create and manage courses
- Upload lectures
- Create tests and questions
- Monitor student performance



# CBT System
- Timed examinations ⏱️
- Multiple-choice questions
- Auto grading system
- Result storage and retrieval
